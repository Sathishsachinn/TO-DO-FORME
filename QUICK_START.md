## 🚀 How to Use

### View Your Calendar
1. Open http://localhost:8000
2. Calendar shows current month with all dates inside the box
3. Hover over holiday symbols to see festival names (desktop)
4. Tap symbols on mobile for notifications

### Add a Task
1. Type task name in "Add a new task..." field
2. Select due date (optional)
3. Select priority (Low/Medium/High)
4. Click "Add" button

### Manage Tasks
- **Click date**: Filter tasks for that day
- **Check mark**: Mark task as completed
- **Delete**: Remove task
- **Search**: Find tasks by name
- **Filter**: Show All/Active/Completed
- **Sort**: By priority, due date, or created date

### Customize
- Toggle dark/light mode with 🌗
- Set privacy password for private tasks
- Export/import tasks as JSON
